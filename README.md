## 🎫 서비스 소개
- 사회적 거리두기로 일상에 제약을 받으면서 관광여행, 스포츠, 문화예술 등 여가활동의 수요 증가
- 반면, 불안한 국제정세로 물가상승세가 이어지면서 중고, 마감할인을 찾는 사람도 증가
#### SORT는 이러한 수요를 반영하여 가성비 좋은 여가생활을 즐길 수 있도록 서비스를 제공함.
